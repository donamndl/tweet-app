# Django Tweet App

A simple Django-based Tweet application where users can create and view tweets with optional image uploads.  
This project was built to practice Django fundamentals such as models, views, templates, forms, and media handling.

---

## Features

- Create tweets using Django ModelForms
- View all tweets in a card-based layout
- Upload and display images with tweets
- Uses Django Template Inheritance
- SQLite database (default Django setup)
